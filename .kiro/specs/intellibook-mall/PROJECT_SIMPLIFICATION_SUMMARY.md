# IntelliBook-Mall 项目简化总结

## 📋 文档更新日期
2025年11月3日

## 🎯 简化目标

将项目从**生产级电商系统**调整为**毕业设计演示系统**，在保证核心功能完整的前提下，优化开发效率和演示便利性。

---

## 🔄 主要变更内容

### 1. 数据库技术栈变更

#### 变更前：MySQL 8.0+
- 需要安装和配置 MySQL 服务器
- 需要创建数据库和用户权限
- 答辩时需要导师配置数据库环境

#### 变更后：SQLite 3
- ✅ 零配置，开箱即用
- ✅ 数据库文件：`intellibook.db`（自动创建）
- ✅ 便携性强，整个数据库就是一个文件
- ✅ 答辩友好，导师可以直接运行项目
- ✅ 支持完整的 SQL 功能和事务处理

**配置变更**：
```properties
# 原配置（MySQL）
spring.datasource.url=jdbc:mysql://localhost:3306/intellibook_mall
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
spring.datasource.username=root
spring.datasource.password=123456

# 新配置（SQLite）
spring.datasource.url=jdbc:sqlite:intellibook.db
spring.datasource.driver-class-name=org.sqlite.JDBC
spring.datasource.username=
spring.datasource.password=
```

### 2. 数据库表结构简化

#### 表数量变更

| 类别 | 原设计 | 简化方案 | 变化 |
|------|--------|----------|------|
| 核心表 | 8 张 | 8 张 | 保留 |
| 辅助表 | 7+ 张 | 0 张 | 删除 |
| **总计** | **15+ 张** | **8 张** | **⬇️ 47%** |

#### 保留的核心表

1. **tb_user** - 用户表（合并了管理员表）
2. **tb_category** - 电子书分类表
3. **tb_ebook** - 电子书表（核心）
4. **tb_order** - 订单表
5. **tb_order_item** - 订单项表
6. **tb_cart** - 购物车表
7. **tb_favorite** - 收藏表
8. **tb_review** - 评价表

#### 删除/合并的表

| 原表名 | 处理方式 | 替代方案 |
|--------|----------|----------|
| tb_mall_user_token | ❌ 删除 | 使用 Session 管理 |
| tb_admin_user | ✅ 合并 | 合并到 tb_user（is_admin 字段） |
| tb_admin_user_token | ❌ 删除 | 使用 Session 管理 |
| tb_ebook_tag | ❌ 删除 | 使用字符串字段（tags） |
| tb_ebook_tag_relation | ❌ 删除 | 使用逗号分隔的 tags 字段 |
| tb_user_behavior | ❌ 删除 | 使用 view_count 字段 |
| tb_reading_progress | ❌ 删除 | 可选功能 |
| tb_download_record | ❌ 删除 | 使用 download_count 字段 |
| tb_review_like | ❌ 删除 | 简化评价功能 |
| tb_carousel | ❌ 删除 | 可选功能 |
| tb_index_config | ❌ 删除 | 可选功能 |
| tb_recommendation_cache | ❌ 删除 | 简化推荐功能 |

### 3. 字段简化

#### tb_ebook 表字段变更

**删除的字段**：
- `original_price` - 合并为单一 price 字段
- `is_deleted` - 改为物理删除
- `create_user` - 简化
- `update_user` - 简化
- `detail_content` - 合并到 book_intro

**新增的字段**：
- `view_count` - 浏览次数（替代用户行为追踪）
- `download_count` - 下载次数（替代下载记录表）

**保留的核心字段**：
- 所有电子书元数据字段（书名、作者、ISBN等）
- 文件管理字段（格式、路径、大小）
- 多语言支持字段（language）
- 评分统计字段（rating、rating_count）

## 💡 开发建议

### 1. 使用 DB Browser for SQLite
- 下载地址：https://sqlitebrowser.org/
- 可视化管理 SQLite 数据库
- 方便查看和编辑数据
- 支持 SQL 查询和导出

### 2. 数据库文件管理
```bash
# 数据库文件位置
intellibook-mall-api/
├── intellibook.db          # SQLite 数据库文件
├── intellibook.db-journal  # 事务日志（自动生成）
```

### 3. 备份和重置
```bash
# 备份数据库
copy intellibook.db intellibook_backup.db

# 重置数据库（删除后重新运行项目自动创建）
del intellibook.db
```

### 4. Git 管理
```gitignore
# .gitignore 中添加
intellibook.db
intellibook.db-journal
*.db
*.db-journal
```
